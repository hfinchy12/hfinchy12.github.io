Network Router Simulator

Assignment completed in CSCE 221: Data Structures and Algorithms
Available on GitHub

Simulated a network packet processing queue in C++.
This program reads in sample network packets and processes the packets into a linked list. This program uses: dynamic memory, linked list, queues, file input, classes, templating, and data filtering. Packets are read in through a text file and then processesd to create a "NetworkPacket" object. Those objects are then sorted based on their content as well as scanned for suspicious input/blocked addresses. A linked list is used as the queue to demonstrate a "first-in first-out" approach to handling data.

Hashing and Simplified Cryptocurrency

Assignment completed in CSCE 221: Data Structures and Algorithms

This program inserts data into a hashtable using a polynomial rolling hash function. Two different implementations of the hashtable are used to avoid collisions, the first being linear probing and the second being a chaining hashtable. The goal for this program is to create an even distribution of collisions.

The second half of this program is to use hashing to create s simplified version of a cryptocurrency blockchain. This applications reads in transactions (strings) from the command line to create a blockchain. The transactions are then hashed and paired with a nonce to validate them according to the rules of the blockchain. The constraint for this blockchain is simple to allow for quick mining method.
